The idea is to add a few inputs/outputs on a Raspberry Pi puppy.

Presentation

Idea

* I want a proper shutdown button for my webserver (Raspi 2) * I want to add some bling bling like blinking a led at each file served * Pilot so display to show message on a led board?

Points of interrest

* Any pin is caoable of GPIO. Not maybe 100% pure hardware (in terms of timing) but jsut any pin will be ok * RPi.GPIO lib can't do hardware PWM, all the PWM are software but at least you can have it on any pin
