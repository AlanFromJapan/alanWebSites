This is the old version of the recent ATmega8515: so build a program for ATmega8515 with ATmel Studio and upload as a AT90S4414 with avrdude.
I got a few of them in my fukubukuro of last new year (Jan2014) at Akizukidenshi and never used them. They look nice, lots of pins, big size (look antique a bit), maybe a bit lightweight regarding program space and memory, but the form factor is great if you need lot of I/O.

Memo fact sheet

  • Code : 4k Bytes / EEPROM : 256 Bytes / RAM : 256 Bytes (half of pin compatible S8515)
  • 4-6v / Max 8Mhz
  • Homepage http://www.atmel.com/devices/atmega8515.aspx
  • Datasheet AT90S4414.pdf

Things to know

* There's just 1 fuse named "fuse" (not lfuse, hfuse... "fuse") * You **can't** read nor program the fuse via Serial Programming: it's in the documentation, and the command aren't in the avrdude.conf. You should write avrdude -p4414 -cavrispmkII -U fuse:w:0xdf:m -v -u but won't work
