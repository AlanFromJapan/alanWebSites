Arduino BT is BlueTooth

Nice tool, recognized by PC ... but can't upload any sketch to it! No idea why, tried many many usual tricks : change port, for small com port number, changed the speed, try other type of card ... no-thi-ng. So my proposal is : let's try with a custom shield that would connect to serial port !!
Update: now I have a Pololu programmer, I can upload code DIRECTLY through the ISP port. Bluetooth is not yet working correctly (you receive text but it's junked), so when I will have time to read the doc I'm sure it can work fine (engineering trick : do it THEN read the doc) >>> Arduino without arduino .
Update: In fact just a combination or Arduino software and Pololu programmer. Build your code with the Arduino IDE (it can't upload anyway), and then upload the .hex file with the Pololu programmer : it works just fine.
