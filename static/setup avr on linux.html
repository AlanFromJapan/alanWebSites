Install

That one is easy: go to the [[https://www.arduino.cc/en/main/software|arduino website]] and download the latest .tar.xz file. You can then //tar -zxf// it or just double click and put the insides somewhere. No need to run the installer unless you want to setup the icons. There, install completed!
**Nota bene:** regarding the //where// to install, I did in /usr/local/apps (which I created). Where to install software in Linux is an eternal debate, so I picked here, but it doesn't matter it will work anywhere.
