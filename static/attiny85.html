ATtiny 85

Memo fact sheet

**Code** : 8192 Bytes / **EEPROM** : 512 Bytes / **RAM** : 512 Bytes
**Homepage** [[https://www.microchip.com/en-us/product/ATtiny85]]

All in all it's beefier than an [[ATtiny2313]] but only in an 8 pins package (THT/SMD).

Fuses

Low/High/Ext (hex): * Default should be: 62/DF/FF * With an external crystal: FE/DF/FF (//avrdude -pt85 -cavrispmkII -v -U lfuse:w:0xfe:m -U hfuse:w:0xdf:m//)
