Some quick commands to remember about **ffmpeg**. It's a fantastic tool, and this is just a memo for me about some commands.

Sample commands

Convert a video to a small gif

ffmpeg -y -i my-input-video.mp4 -filter_complex "fps=10,scale=300:-1:flags=lanczos,split[s0][s1];[s0]palettegen=max_colors=32[p];[s1][p]paletteuse=dither=bayer" output.gif

Memo

Start time

Use the //-ss 00:00:03// parameter to start at 3 sec (put just after the input name).

Remember me

* Parameters order matter, so ffmpeg knows how you want to process things in which order, if applies to input stream or output stream
